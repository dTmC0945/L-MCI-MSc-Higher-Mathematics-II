This document includes the contents of Higher Mathematics II, official name being Höhere Mathematik 2, taught at MCI in the Mechatronik Smart Technologies. This document is the part of the module MECH-M-2-HMA-HMA-ILV taught in the M.Sc degree.

All relevant code of the document is done using SageMath where stated using v10.3 and Python v3.13.1.

This document was compiled with LuaTeX v1.18.0, and all editing were done using GNU Emacs v29.4 using AUCTeX and org-mode package.
